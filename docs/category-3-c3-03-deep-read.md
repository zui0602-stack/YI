## 架構收斂

- 數值與 OCR：9232、22406、22269 和 742 可合併為「OCR 文字 -> 結構化欄位 -> 型別轉換 -> 條件判斷」管線。

- 視覺識別與手勢：8630、9408、15515、897、4578 補齊圖像定位、濾鏡調參、驗證類風險與滑動搜尋。
- 工程與環境：5903、17276、21876、20221 補齊安全更新、intent、入門安裝與網路設定邊界。

## 安全邊界

- 滑塊驗證、圖片驗證、圖形驗證只保留授權 UI 測試與 OCR/濾鏡校準概念，不提供第三方驗證繞過手冊。
- 隨機中文字符不整理成批量註冊或平台濫用方案，只保留測試資料生成思路。
- 自動更新只保留 manifest、版本檢查、簽名/校驗、回滾與脫敏原則，不收入可替換本地腳本的完整源碼。
- IP/網路配置只記錄環境恢復與診斷，不整理代理切換、規避限制或平台對抗流程。
