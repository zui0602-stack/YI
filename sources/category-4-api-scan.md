## 范围规则

- 使用者在 2026-06-15 更新范围：分类 4 交流区只学习 2024 年之后的内容。
- 当前采用的有效范围：文章建立时间 `created_at >= 2024-01-01`。
- `created_at < 2024-01-01` 的分类 4 旧交流帖暂不纳入 GitHub 学习笔记，即使它们被 2024 后回复顶到前面，也不作为学习来源。
- 分类 4 回复楼层也套用同一原则：只吸收 2024-01-01 后的有效回复。
- 例外：分类 3 是正式教程区，不套用这个日期排除规则。

## API 发现

论坛前端使用的接口：

- 文章列表：`https://bbs.zdjl.site/bbs/threads/list`
- 回覆列表：`https://bbs.zdjl.site/bbs/posts/list`
- 必要请求头：`mbbs-domain: zdjl.mbbs.cc`
- 分类参数：`category_id=4`
- 分页参数：`page_offset`, `page_limit`
- 排序参数：`sort=-posted_at`

这个接口让分类 4 可以完整建索引，不再受网页滚动只能加载一部分的限制。

## 后续处理规则

- 分类 4 不整篇搬运到 GitHub，只整理成原创的“问题模式、推荐做法、适用限制、来源链接”。
- 有回覆的文章必须看 `posts/list` 楼层，很多真正答案不在主文。
- 涉及卡密、后端、登录、热更新、密钥或第三方接口时，只整理架构与安全注意，不复制敏感完整代码。
- 视频线索需要二次确认：区分“教学影片”与“视频处理脚本场景”。
